# AWSUME_FZF [AKA: AWSELECT]

FuziFind on your `awsume -l` list and select one of yout AWS profiles.

## How to install

```bash
brew tap mariogarridopt/awsume-fzf
brew install awselect
```

## How to use

```bash
awselect
```
